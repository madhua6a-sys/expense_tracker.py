# expense_tracker.py
A Python-based CLI Expense Tracker that allows users to add, view, and delete daily expenses. The project uses JSON file handling to store expense data permanently and helps users manage expenses through a simple menu-driven interface.

## Features

- Add expenses
- View expenses
- Delete expenses
- Store data using JSON file handling

## Technologies Used

- Python
- JSON
- File Handling

## Example

1. Add expense
2. View expenses
3. Delete expense
4. Exit

## Concepts Used

- Functions
- Dictionaries & Lists
- CRUD Operations
- Exception Handling
- File Handling
